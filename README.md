# 基于ZYNQ Linux CAN通讯开发资源文件

## 简介

本资源文件提供了基于ZYNQ Linux平台的CAN通讯开发相关内容。资源包括Vivado端的PS-PL配置以及SDK端的例程，帮助开发者快速上手并实现ZYNQ平台上的CAN通讯功能。

## 内容概述

1. **Vivado端配置**
   - 详细介绍了如何在Vivado中配置ZYNQ的PS（Processing System）和PL（Programmable Logic）部分，以支持CAN通讯功能。
   - 提供了配置步骤和相关设置的截图，确保开发者能够正确配置硬件平台。

2. **SDK端例程**
   - 提供了基于SDK的CAN通讯例程代码，展示了如何在ZYNQ Linux环境下进行CAN通讯的开发。
   - 代码中包含了初始化、数据发送和接收等基本操作，帮助开发者理解CAN通讯的实现过程。

## 使用说明

1. **Vivado配置**
   - 按照资源文件中的步骤，在Vivado中完成PS-PL的配置。
   - 确保硬件平台支持CAN通讯功能。

2. **SDK例程**
   - 将提供的SDK例程导入到SDK环境中。
   - 根据实际硬件配置，调整代码中的相关参数。
   - 编译并运行例程，验证CAN通讯功能。

## 注意事项

- 在运行SDK例程时，注意检查CAN总线的连接状态，确保通讯正常。
